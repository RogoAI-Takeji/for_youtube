\# 🎙️ ROGOAI Voice Studio



&nbsp;



\*\*Universal Voice Generation Platform\*\*

VOICEVOXのキャラクター音声と、Coqui TTSによる自分の声のクローン生成を、1つの画面で操作できる統合ツールです。



-----



\## ⚠️ 重要：最初にお読みください (Important Notice)



このアプリは高機能なAIエンジンを搭載しているため、\*\*初回起動時のみ\*\* 以下の準備が必要です。



\### 1\\. VOICEVOXの準備 (必須)



本アプリはVOICEVOXのエンジンを利用します。



&nbsp; \* \*\*事前に \[VOICEVOX公式サイト](https://voicevox.hiroshiba.jp/) からアプリをダウンロードし、起動しておいてください。\*\*

&nbsp; \* VOICEVOXが起動していないと、キャラクター音声は生成できません。



\### 2\\. 初回起動には時間がかかります



初回起動時、Coqui TTSのAIモデル（約2GB）やPyTorchなどのライブラリを自動ダウンロードします。



&nbsp; \* \*\*所要時間:\*\* ネット環境により \*\*15分〜30分程度\*\* かかる場合があります。

&nbsp; \* \*\*画面:\*\* 黒い画面（コンソール）でダウンロードが進みますので、閉じずにお待ちください。

&nbsp; \* \*\*2回目以降:\*\* すぐに起動します。



\### 3\\. Windowsセキュリティの警告について



個人の開発者が作成した署名なしアプリのため、Windows SmartScreenによりブロックされる場合があります。



&nbsp; \* 青い画面が出た場合： \*\*「詳細情報」→「実行」\*\* をクリックして起動してください。ウイルス等は含まれておりません。



-----



\## ✨ 主な機能



&nbsp; \* \*\*🎭 デュアルエンジン統合\*\*

&nbsp;     \* \*\*VOICEVOX:\*\* ずんだもん、四国めたん等のキャラクター音声を簡単生成。

&nbsp;     \* \*\*Coqui TTS (XTTS):\*\* たった数秒の音声ファイルから、その人の声をクローンして喋らせることが可能。

&nbsp; \* \*\*🌍 多言語対応 (Coqui TTS)\*\*

&nbsp;     \* 日本語、英語、中国語、韓国語、フランス語、ドイツ語など多言語で発話可能。

&nbsp; \* \*\*📁 便利なファイル出力\*\*

&nbsp;     \* ファイル名に「連番」「日時」「テキストの一部」などを自動付与するカスタム命名機能。

&nbsp;     \* WAV / MP3 出力対応。

\## 📦 インストールと起動 (Installation)



\### 必要動作環境



&nbsp; \* \*\*OS:\*\* Windows 10 / 11 (64bit)

&nbsp; \* \*\*CPU:\*\* Intel Core i5 以上推奨

&nbsp; \* \*\*GPU:\*\* NVIDIA GeForce 推奨（なくてもCPUモードで動作します）

&nbsp; \* \*\*メモリ:\*\* 8GB以上推奨



\### 手順



1\.  \*\*\[Releasesページ](https://github.com/RogoAI-Takeji/for\_youtube/releases)\*\* から最新のZIPファイルをダウンロードします。

2\.  任意の場所に解凍します。

&nbsp;     \* \*\*⚠️注意:\*\* 解凍後のフォルダにある `user\_data` フォルダは削除するか、空にしておいてください（他者の設定が残っているのを防ぐため）。

3\.  \*\*`RogoAI\_Voice\_Studio\_v1.9.exe`\*\* (または起動バッチファイル) をダブルクリックして起動します。

\## 🚀 使い方 (Usage)



\### モード切替



画面上部のラジオボタンで、使用するエンジンを切り替えます。



\#### 🔹 VOICEVOXモード



1\.  VOICEVOXアプリを別途起動しておきます。

2\.  「再接続」ボタンを押し、接続を確認します。

3\.  キャラクターを選択し、テキストを入力して「生成開始」！



\#### 🔹 Coqui TTSモード



1\.  「話者ファイル」にお手持ちの音声ファイル（wav/mp3）を指定します。

2\.  喋らせたい言語を選択します（日本語=ja）。

3\.  テキストを入力して「生成開始」！

&nbsp;     \* ※GPUがない場合、生成に少し時間がかかります。
