\# Metadata Scan \& Clean



画像や動画ファイルのメタデータ（個人情報やAI生成パラメータ）を検知し、削除するためのWindowsデスクトップアプリです。



!\[Version](https://img.shields.io/badge/version-1.0.0-blue.svg)

!\[License](https://img.shields.io/badge/license-MIT-green.svg)



\## 概要 (Overview)

このアプリは、指定したフォルダ内の画像や動画をスキャンし、プライバシーリスクのあるメタデータ（GPS位置情報、撮影者情報、AI生成プロンプトなど）を削除します。



SNSや共有サイトへアップロードする前の「安全確認」に最適です。



\## 特徴 (Features)

他のメタデータ削除ツールとの違いは、\*\*「AI生成画像への対応」\*\*です。



\* \*\*基本的なメタデータの削除:\*\*

&nbsp;   \* GPS位置情報（Location data）

&nbsp;   \* Exif情報（撮影日時、カメラ機種など）

&nbsp;   \* 著作者情報（Author, Copyright）

\* \*\*AI生成情報の削除（独自機能）:\*\*

&nbsp;   \* ComfyUIやWebUIなどで生成された画像に含まれる「プロンプト」や「ワークフロー」情報を検知・削除します。

\* \*\*2つのクリーニングモード:\*\*

&nbsp;   \* \*\*スマート削除（推奨）:\*\* 画質を劣化させずにメタデータのみを除去します。

&nbsp;   \* \*\*完全削除:\*\* ファイルを再エンコードして、全ての付加情報を完全に消去します。



\## 使い方 (Usage)



1\.  \*\*起動:\*\* `MetadataScanClean\_JP.exe` を起動します。

2\.  \*\*フォルダ選択:\*\* 「参照」ボタンをクリックして、チェックしたいフォルダを選びます。

3\.  \*\*スキャン:\*\* 「🔍 スキャン」ボタンを押すと、危険なファイルがリストアップされます。

&nbsp;   \* 🔴 赤色アイコン: GPSなどの重要情報が含まれています。

&nbsp;   \* 🟡 黄色アイコン: AI生成情報などが含まれています。

4\.  \*\*実行:\*\* 「✨ 実行」ボタンを押すと、元のフォルダの隣に `\_clean` という名前のフォルダが作成され、そこに安全なファイルが出力されます。
