# Metadata Scan & Clean

画像や動画ファイルのメタデータ（個人情報やAI生成パラメータ）を検知し、削除するためのWindowsデスクトップアプリです。

![Version](https://img.shields.io/badge/version-2.0.1-blue.svg)
![License](https://img.shields.io/badge/license-MIT-green.svg)

## 概要 (Overview)

このアプリは、指定したフォルダ内の画像や動画をスキャンし、プライバシーリスクのあるメタデータ（GPS位置情報、撮影者情報、AI生成プロンプトなど）を削除します。

SNSや共有サイトへアップロードする前の「安全確認」に最適です。

## 特徴 (Features)

他のメタデータ削除ツールとの違いは、**「AI生成画像への対応」**です。

* **基本的なメタデータの削除:**
    * GPS位置情報（Location data）
    * Exif情報（撮影日時、カメラ機種など）
    * 著作者情報（Author, Copyright）
* **AI生成情報の削除（独自機能）:**
    * ComfyUIやWebUIなどで生成された画像に含まれる「プロンプト」や「ワークフロー」情報を検知・削除します。
* **2つのクリーニングモード:**
    * **スマート削除（推奨）:** 画質を劣化させずにメタデータのみを除去します。
    * **完全削除:** ファイルを再エンコードして、全ての付加情報を完全に消去します。

## 📁 対応ファイル形式 (Supported File Formats)

本アプリは以下のファイル形式に含まれるメタデータを自動検出し、クリーニングします。

### 画像 (Image)
```
jpg / jpeg / png / webp / gif / tif / tiff
```

### 動画 (Video)
```
mp4 / mov / avi / mkv / webm / m4v
```

### 音声 (Audio)
```
wav / mp3 / flac
```

これらの形式に対しては、GPS・Exif・Author情報・AI生成プロンプト（ComfyUI/SD/WebUI）などの危険メタデータを削除した安全なコピーを `_clean` フォルダに出力します。

## 使い方 (Usage)

1. **起動:** `MetadataScanClean_JP.exe` を起動します。
2. **フォルダ選択:** 「参照」ボタンをクリックして、チェックしたいフォルダを選びます。
3. **スキャン:** 「🔍 スキャン」ボタンを押すと、危険なファイルがリストアップされます。
    * 🔴 赤色アイコン: GPSなどの重要情報が含まれています。
    * 🟡 黄色アイコン: AI生成情報などが含まれています。
4. **実行:** 「✨ 実行」ボタンを押すと、元のフォルダの隣に `_clean` という名前のフォルダが作成され、そこに安全なファイルが出力されます。

## ⚠️ 重要：スキャン対象外ファイルの扱い (Important: Unsupported File Handling)

### 安全設計の原則

本アプリは**安全性を最優先**するため、スキャン対象外のファイル（例：`txt`, `pdf`, `docx`, `xlsx`, `zip`, `json` など）は `_clean` フォルダへ**自動コピーしません**。

### なぜコピーしないのか？

1. **`_clean` フォルダは "100% メタデータが除去された安全ファイルだけが入る領域"** として扱います。
2. スキャン不可のファイルをそのまま `_clean` にコピーすると、ユーザーが **「_clean にあるから安全だろう」** と誤認し、PDFの "作成者名（本名）" などの個人情報が流出する危険があります。
3. セキュリティツールにおいては、**「手動でコピーする」行為がユーザーの注意喚起となり、事故防止**になります。

### 📝 対応方法（ユーザー向け案内）

テキストファイル・PDF・資料など、クリーニング不要のファイルを同梱したい場合は、**クリーニング完了後、手動で `_clean` フォルダへコピー**してください。

この設計により、**"_clean = 100% セーフゾーン"** を堅持し、誤解による個人情報流出を確実に防ぎます。
