\# 使い方ガイド (User Guide)



Metadata Scan \& Clean の詳しい使い方です。



\## 1. 準備

1\. GitHubの「Releases」ページから、ZIPファイルをダウンロードします。

2\. ZIPファイルを右クリックして「すべて展開」を選びます。

3\. 展開されたフォルダの中にある `.exe` ファイルをダブルクリックして起動します。



> \*\*※警告が出た場合\*\*

> Windows Defenderにより「WindowsによってPCが保護されました」と表示されることがあります。

> これは個人の開発者が作ったアプリでよく出る表示です。

> 「詳細情報」→「実行」をクリックすると起動できます。



\## 2. スキャンする（確認）

1\. \*\*「参照」\*\* ボタンを押して、チェックしたい画像が入っているフォルダを選びます。

2\. \*\*「🔍 スキャン」\*\* ボタンを押します。

3\. 画面にファイルの一覧が表示されます。

&nbsp;   \* 🔴 \*\*赤色\*\*: GPS（位置情報）や個人名が含まれています。

&nbsp;   \* 🟡 \*\*黄色\*\*: AI生成情報（プロンプトなど）が含まれています。



\## 3. クリーニング（削除）

1\. 削除モードを選びます。

&nbsp;   \* 基本的には \*\*「スマート削除（推奨）」\*\* のままでOKです。

2\. \*\*「✨ 実行」\*\* ボタンを押します。

3\. 元のフォルダの隣に `\_clean` という名前のフォルダが新しく作られ、そこに安全なファイルが保存されます。



\## 4. ビフォー・アフター確認

1\. \*\*「ビフォー・アフター」\*\* タブを開きます。

2\. \*\*「📂 元ファイル選択」\*\* で、処理前のファイルを選びます。

3\. メタデータが綺麗に消えていることを確認できます。
