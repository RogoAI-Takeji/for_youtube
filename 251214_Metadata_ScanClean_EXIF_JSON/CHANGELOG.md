# Change Log

このプロジェクトの更新履歴を記録します。

## [v2.0.1] - 2025-12-02

### Fixed (不具合修正)

* **コンソールウィンドウの点滅を防止:**
    * EXE化して実行した際、動画や音声ファイルの解析・処理中（FFmpeg/FFprobe呼び出し時）に、PowerShellやコマンドプロンプトの黒い画面が一瞬表示されては消える現象を修正しました。
    * バックグラウンドプロセスとして完全に無音・無表示で動作するように改善し、他の作業への干渉を防ぎました。

## [v2.0.0] - 2025-12-02

### Added (追加)

* **多言語サポート:**
    * 内部設定により、日本語 (JP) と英語 (EN) の切り替えに対応しました。
* **ビルド構成の変更:**
    * 日本語版 (`_JP.exe`) と英語版 (`_EN.exe`) を個別にビルドできるように構造を変更しました。

### Changed (変更)

* **UI調整:**
    * 英語テキストの表示に合わせて、ウィンドウ幅を 500px から 550px に拡張しました。

## [v1.0.0] - 2025-11-27

### Added (追加)

* 初回リリース
* 画像（GPS/Exif/AI生成情報）および動画・音声のメタデータ削除機能を実装。
* スマート削除モードと完全削除モードの実装。
